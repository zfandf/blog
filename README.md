# [Fannie](http://blog.fflove.top)的个人博客源码.

## hexo 快速安装

### 安装前提
* [NodeJs](http://nodejs.org/)
* [GIT](http://git-scm.com/)

### 安装
    npm install -g hexo-cli
### 建站
* 创建站点编码文件夹
        
        midkr blog 
    
* 初始化文件夹
        
        hexo init blog
        
* npm 安装插件
        
        npm install
        
* 启动服务

        hexo server
        
* 发布
        
        hexo p
